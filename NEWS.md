microbes 0.5
===========

* new `str_wrap` function which gives `strwrap` output in a more
  convenient format
  
  You can use devtools::show_news() to display the NEWS using R’s built-in parser 
  and check that it appears correctly. show_news() defaults to showing just the 
  news for the most recent version of the package. You can override this by using 
  argument latest = FALSE.
